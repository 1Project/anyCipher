# anyCipher
<h4>A simple Java program with miscellaneous cipher algorythms implementation.</h4>

<ul>
  <li>Book</li>
  <li>Cezar</li>
  <li>Modulus gamma</li>
  <li>Slogan</li>
  <li>Table</li>
  <li>Vertical</li>
  <li>Permutation</li>
  <li>Vigenere</li>
</ul>

To change it to english alphabet you only have to update ALPHABET constant in CypherHelper class.
